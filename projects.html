Projects

Donation Platform

Highly scalable donation platform with Docker + GKE, RabbitMQ, MongoDB; autoscaling and strong monitoring/security.

Architecture designed for high scalability and availability using Docker for standardized environments and Google Kubernetes Engine (GKE) for orchestration with autoscaling. RabbitMQ ensures reliable messaging and MongoDB provides scalable data storage. Load testing showed ~30,000 requests/second. Monitoring with Prometheus and Grafana; security via OWASP ZAP, SonarQube, and Checkmarx (DAST/SAST).

Digital Production Scraping using Java

Large-scale web scraping on AWS with API Gateway, ALB, EC2 ASG, RabbitMQ, MongoDB (replicas); k6-verified performance.

Built to handle large data volumes and high traffic: API Gateway for secure access, Load Balancer distributing to EC2 instances with Auto Scaling. RabbitMQ for async communication, MongoDB with read replicas for performance. Load testing with k6 validated ~1,500 requests/second. Monitoring with Prometheus and Grafana. Security: DAST (OWASP ZAP) and SAST (SonarQube, Checkmarx).

Application Migration (GCP to AWS)

End-to-end migration of catch.app from GCP to AWS using DMS/SMS; improved performance and reduced costs.

Source on GCP included compute, DBs, storage and security configurations. Target AWS architecture used EC2, S3, and RDS with optimized configs for performance and scalability. Phased migration (data, application, infrastructure) using AWS Database Migration Service and AWS Server Migration Service delivered a smooth, cost-efficient transition and better performance.

Liveauctions

Real-time app on GCP for massive concurrent connections using LB + Compute Engine; RabbitMQ (Shovel) + WebSTOMP.

Highly scalable live application with GCP Load Balancer across elastic compute instances. MySQL as primary DB. RabbitMQ with Shovel for robust messaging and WebSTOMP for efficient real-time communication. Load testing with k6 and LoadView validated up to ~100,000 concurrent connections. Monitoring via Grafana; security validated with DAST (OWASP ZAP) and SAST (SonarQube, Checkmarx).
